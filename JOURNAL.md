Alright, so I’m finally starting this project for Stasis! I'm officially calling it the PiPixel. The absolute dream is to build a pocket-sized, retro-looking digital camera (kind of like an old Sony Cyber-shot style layout). I want a custom 3D-printed case, a display screen on the back so I can scroll through my photos and check out what I shot.

Spent hours today just comparing camera sensors. I almost went with a cheaper one, but ended up choosing the Raspi Cam 3 because it actually has autofocus, which is a total lifesaver. For the main board, I’m using the Pi Zero 2W. Honestly, it’s the absolute bare minimum that can handle a live camera stream without turning into a total slideshow, but it’s tiny and perfect for this form factor.

Power delivery was a whole rabbit hole. I initially wanted to use a super thin 1500mAh LiPo battery paired with a TP4056 charger and an MT3608 boost converter to get a steady 5V. But i had to change to using 2S Standard 18650 battery  2500mah Cuz I had power the Pi, TFT LCD, CAM and RGBA leds for a min 2hrs. For the screen, I’m using an ILI9341 Non Touch 2.8" display, and I've got a ESP32 which i will be using for Remote Control. I’m also adding a WS2812B RGB LED for status indications. I’m deliberately avoiding a custom PCB because I want to be flexible with where I stuff the components inside the case.
